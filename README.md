# 감정 전염 설문 웹페이지

## 가장 쉬운 실행 방법

`index.html`을 더블클릭하면 됩니다. 현재 버전은 CSS, JavaScript, 표정 자극 이미지가 모두 HTML 안에 포함된 단일 파일입니다. 따라서 `index.html` 하나만 복사해도 보라색 웹앱 UI가 그대로 작동합니다.

## GitHub Pages

저장소 최상위에 `index.html`을 업로드하고 Settings → Pages → Deploy from a branch → main / root를 선택합니다. 이전 파일이 남아 있으면 저장소의 기존 `index.html`을 삭제한 뒤 이 파일로 교체하고, 브라우저에서 Ctrl+F5로 강력 새로고침합니다.

## 확인 방법

페이지 소스에서 `PURPLE WEBAPP BUILD 2026-07-11 SINGLE-FILE` 문구가 보이면 최신 파일입니다.
